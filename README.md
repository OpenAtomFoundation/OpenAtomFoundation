# 关于我们

[开放原子开源基金会](https://openatom.cn/)是致力于推动全球开源事业发展的非营利机构，于2020年6月在北京成立。基金会秉持“共建、共治、共享”的核心理念，定位为以开发者为中心的开源项目孵化平台和科技公益服务机构，旨在系统打造开放协同的开源生态，提升行业协作效率，赋能千行百业，推动我国在全球开源领域的影响力不断提升，助力建设繁荣的开源生态体系。

## 开源项目

### 已归档项目

>声明：所有开源项目都有其生命周期，并且可能因多种原因变得不够活跃。在开放原子开源基金会内部，项目处于孵化阶段需要满足多项要求，按照开放原子开源基金会开源项目管理制度v1.0第三章第十二条中指出，孵化期项目或毕业项目存在以下情形之一的，秘书处将对其进行归档：<br>
> （一）开源项目维护团队解散的；<br>
> （二）项目方或开源项目治理组织向秘书处发起项目归档提案的；<br>
> （三）开源项目一年内无代码提交或维护操作且经通知后仍未维护的。<br>
> 项目一旦被归档：开放原子开源基金会将不再为该项目提供全面服务，如品牌营销、开发者活动等。已归档项目的商标及代码仍由开放原子开源基金会以中立方式托管，项目代码仓设置为只读状态。如有志人士希望接管项目运营，重启项目，请联系<operation@openatom.org>
